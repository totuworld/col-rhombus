# col-rhumbus
Local LLM을 활용한 Chrome Extension

## 주요 기능
* 페이지 요약 기능
* 페이지 내용을 기반으로 Local LLM과 질의 응답

## 개발 방법
1. `pnpm install`을 통해 의존성 설치
2. `pnpm dev`로 실행
3. `chrome://extensions/`에서 개발자 모드 활성화 후 `dist` 폴더를 로드
